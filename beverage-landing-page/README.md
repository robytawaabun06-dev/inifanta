# Beverage Landing Page

## Overview
This project is a landing page for a beverage product, designed to provide users with an engaging and informative experience. The website features a clean and professional layout using orange and white colors.

## Project Structure
The project consists of the following files and directories:

```
beverage-landing-page
├── assets
│   ├── images          # Directory for images used in the website
│   └── videos          # Directory for video files related to the product
├── css
│   └── style.css       # Stylesheet for the website
├── js
│   └── script.js       # JavaScript file for functionality
├── bootstrap
│   └── bootstrap.min.css # Local Bootstrap CSS for responsive design
├── index.html          # Home page of the website
├── product.html        # Product page featuring a video and description
├── about.html          # About brand page with brand information
├── contact.html        # Contact page serving as the footer
└── README.md           # Documentation for the project
```

## Setup Instructions
1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Open `index.html` in your web browser to view the home page.
4. Use the navigation links to explore the product, about, and contact pages.

## Technologies Used
- HTML
- CSS
- JavaScript
- Bootstrap (for responsive design)

## Features
- Responsive design that adapts to different screen sizes.
- Engaging layout with images and videos to showcase the beverage product.
- Clear navigation between different sections of the website.
